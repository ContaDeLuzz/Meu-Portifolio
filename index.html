Olá, me chamo ContaDeLuz e gosto de programação. Acompanhe alguns de meus projetos que demonstram minha paixão por criar experiências interativas e visualmente atraentes.

Habilidades

HTML

Linguagem de marcação utilizada para estruturar e organizar o conteúdo de páginas web. Com HTML, crio a base de todos os meus projetos web, garantindo uma estrutura semântica e acessível para diferentes dispositivos e usuários.

CSS

Linguagem de estilização que utilizo para dar vida aos meus projetos web. Com CSS, crio layouts responsivos, animações fluidas e interfaces visualmente atraentes que proporcionam uma experiência agradável aos usuários em qualquer dispositivo.

JavaScript

Linguagem de programação que me permite adicionar interatividade e funcionalidades dinâmicas aos meus projetos. Com JavaScript, desenvolvo desde simples animações até aplicações web complexas, criando experiências interativas e responsivas.

Scratch

Plataforma de programação visual que utilizo para criar jogos, animações e histórias interativas. Com Scratch, posso desenvolver projetos criativos de forma rápida e intuitiva, explorando conceitos de lógica de programação através de blocos visuais.
